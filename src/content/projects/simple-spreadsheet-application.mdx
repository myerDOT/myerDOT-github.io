This simple spreadsheet application supports all basic arithmetic operators. It also supports these aggregate operators:
- Average (AVE)
- Median (MEDIAN)
- Maximum (MAX)
- Minimum (MIN)
- Count (COUNT)
- Count all (COUNTA)
- Sum (SUM)

...and these unary operators:
- Absolute value (ABS)
- Increment (++)
- Decrement (--)
- Negation (NEG)
